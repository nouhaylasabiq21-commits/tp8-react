# TP – Consommer une API avec React

## 📌 Objectif

Ce TP a pour objectif d’apprendre à consommer une API dans une application React en utilisant deux méthodes :

* `fetch()` (JavaScript natif)
* `axios` (bibliothèque externe)

---

## 🚀 Fonctionnalités réalisées

### ✅ Récupération de données

* Appel à une API publique (`jsonplaceholder`)
* Récupération des utilisateurs et des articles

### ✅ Méthodes utilisées

* `fetch()` pour récupérer 10 articles
* `axios` pour récupérer les utilisateurs

### ✅ Affichage

* Liste des articles (titre)
* Liste des utilisateurs (nom, email, ville)

### ✅ Gestion des états

* Chargement (`loading`)
* Erreur (`error`)

### ✅ Interaction

* Bouton **"Recharger les données"** pour relancer les requêtes

## 🧠 Concepts clés

* **API** : source de données externe
* **fetch** : méthode native pour les requêtes HTTP
* **axios** : alternative plus simple pour les requêtes
* **useEffect** : exécuter du code au chargement
* **useState** : gérer l’état des données
